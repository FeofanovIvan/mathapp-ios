# MathApp iOS

Образовательное приложение для подготовки к ЕГЭ/ОГЭ по математике. Помогает школьникам тренировать задачи в игровом формате — с персонажами-математиками и системой прогресса.

[![App Store](https://img.shields.io/badge/App%20Store-Download-blue?logo=apple)](https://apps.apple.com/app/mathup/id6502262530)

## ✨ Возможности

- Задачи по математике уровня ЕГЭ и ОГЭ
- Персонажи — реальные математики (Гаусс, Тьюринг, Ньютон и др.)
- Система прогресса и статистики
- Оффлайн-режим на CoreData
- SwiftUI-интерфейс с анимациями

## 🛠 Стек

- **Swift** + **SwiftUI**
- **CoreData** — хранение прогресса
- **Firebase** — аналитика
- Архитектура: MVVM

